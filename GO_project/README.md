This folder contains all the inputs, raw outputs and scripts necessary to replicate the results in this study

All cytoscape- and R-based enrichment analyses were conducted in a computational platform containing an AMD Ryzen 5 4600G 3.7 GHz 11MB Cache AM4 and 16GB RAM DDR5.
Platforms with weaker specifications should also be able to run such programs with no particular issues, but 16GB RAM is recommended.
